# Producto Smartwatch

Proyecto de sitio estático que presenta el diseño de una página de producto. Incluye HTML, CSS y JavaScript mínimo, además de imágenes de producto en la carpeta `assets/images`.

## Puedes verlo en:

https://peperigores.github.io/producto-smartwatch/ 

## Si no estuviera disponible:
- Descarga y descomprime el archivo .zip o clona el repositorio.
- Abre el archivo [index.html](index.html) en tu navegador para ver la página.

## Características
- Página estática lista para mostrar un producto.
- Estilos en [css/styles.css](css/styles.css).
- Interactividad mínima en [js/main.js](js/main.js).
- Imágenes optimizadas en [assets/images](assets/images).

## Estructura del proyecto
- [index.html](index.html) — Página principal.
- [css/styles.css](css/styles.css) — Estilos.
- [js/main.js](js/main.js) — Scripts.
- [assets/images](assets/images) — Imágenes del producto.

## Instalación y uso
1. Clona el repositorio o descarga los archivos.
2. Abre [index.html](index.html) en tu navegador (doble clic o arrastrar al navegador).


## Buenas prácticas
- Mantén imágenes optimizadas (AVIF/WebP preferible).
- Usa herramientas de build si vas a escalar (parcel, vite, etc.).
